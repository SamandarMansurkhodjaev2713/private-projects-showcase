# Project Card Gallery

Shareable bilingual one-page cards for selected projects by **Samandar Mansurkhodjaev**.

These cards are designed for fast review by recruiters, tech leads, founders and clients. Each card explains the product, the stack, the architecture, the reason behind the architecture and the strongest professional signal without exposing private source code or sensitive business logic.
