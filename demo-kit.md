# Demo & Interview Kit

Практичный сценарий, что показывать на интервью, созвоне с клиентом, разговоре с техлидом или презентации партнеру.

## Правила безопасного показа

- Не показывать приватный source code, `.env`, tokens, client data, internal dashboards с чувствительными данными.
- Показывать архитектуру, live demos, публичные репозитории, GitHub Actions, обезличенные screenshots и карточки проектов.
- Для приватных проектов объяснять роль, стек, решения, tradeoffs и business value без раскрытия коммерческой логики.
